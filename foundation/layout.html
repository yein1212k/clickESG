Layout

화면에 있는 요소들의 시각적 배열을 다양한 디바이스에서도 일관성을 유지할 수 있도록 도와줍니다.

Responsive layout grid

Break point
  • 우리나라에서 가장 많이 사용되는 데스크탑, 태블릿, 모바일 크기에 기반한 브레이크 포인트 입니다.
  • 연합뉴스는 반응형 웹으로 모바일일 경우를 최우선으로 초점을 맞춰서 디자인해줍니다.
Break point
디바이스 브레이크 포인트(px) 대표화면 사이즈(px)
데스크톱 1280 ~ 1920
데스크톱 1024 ~ 1279 1024
태블릿 768 ~ 1023 768
모바일 320 ~ 767 375
Contents area
  • 다양한 디바이스에서 보이는 화면의 최대 콘텐츠 영역을 설정하여 사용합니다.
  • 반응형을 기반으로 레이아웃을 구성하며 디바이스 축소 시에는 콘텐츠가 재배치됩니다.

Spacing

균일한 요소와 간격을 사용하여 UI의 체계적인 배열을 도와줍니다.
8px(8-Point grid)의 배수를 사용하며, 상황에 따라 4px의 배수 간격을 사용할 수 있습니다.
