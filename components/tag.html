Tag

키워드 또는 레이블을 사용하여 콘텐츠를 분류하는 수단입니다. 콘텐츠 항목에 직접 관련 분류 체계, 데이터 속성을 표시하거나,
목록에서 특정 분류 체계, 데이터 속성을 가진 항목이 선택되었음을 보여주기 위한 태그 그룹으로 사용됩니다.

Anatomy

Options

Label and Icon

탭 항목에는 접근성을 위한 레이블이 있어야 합니다.

Keyword

페이지 내에서 중요도의 계층을 만드는 데 사용해야 합니다.
Depth의 숫자가 작을수록 중요도가 높아집니다.

Icon

선택된 상태의 탭 항목은 현재 탭 항목을 보여줍니다.
한 번에 탭 항목 하나만 선택할 수 있습니다.

group

탭 항목 컨테이너로 탭의 영역을 나타냅니다.
사용자가 클릭 가능한 범위로 설정 되어야 합니다.

Text overflow

태그 텍스트가 사용 가능한 수평 공간에 비해
너무 길면 말줄임 표시로 줄여줍니다.
