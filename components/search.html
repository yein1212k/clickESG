Search

사용자가 검색을 통해 원하는 콘텐츠를 빠르게 찾을 수 있게 해주는 컴포넌트 입니다.
검색창은 단어 입력시 자동완성 기능 및 유사 검색어를 통해 사용자가 원하는 검색어를 보여줍니다.

Anatomy

Options

Label and Icon

검색어를 입력하기 전 검색 필드에는
레이블과 검색 아이콘이 포함되어야 합니다.

Typing

텍스트 필드를 활성화한 후 입력을 입력합니다.

Value

사용자가 입력한 텍스트를 일반 본문 텍스트 스타일로 표시합니다.

Close icon

입력 값을 한번에 지울 수 있는 아이콘입니다.

Width

너비는 상황에 맞게 적절히 사용자 지정할 수 있습니다.

Used for GNB

GNB 검색 유틸을 눌렀을 때 나오는 검색 창 유형입니다.

Behaviors

Overflow

입력한 텍스트가 필드의 가로 공간에 비해 너무 길면 텍스트가 잘립니다.

Responsive layout

레이아웃을 확장할 때 검색 창 위치와 정렬을 어떻게 확장해야 하는지 고려해서 디자인 해야 합니다.

모바일에서는 전체 화면 넓이로 노출하며, 중간 창과 확장된 창에서는 지정된 크기로 노출합니다.

실시간 인기 검색어 노출

검색어 입력 전에 노출되는 실시간 인기 검색어 키워드입니다.

연관 검색어(자동완성)

검색어 입력 후에 나오는 연관 검색어로 입력된 검색어가 있는 경우 연관 검색어 영역만 노출됩니다.

최근 검색어

검색어 입력 전에 최근 검색어가 있는 경우
최대 5개 까지 노출됩니다.

Usage

오류 대신 "결과 없음" 상태 표시
  • 검색 필드에는 오류 상태가 없습니다.
  • 오류를 표시하는 대신 "결과 없음" 페이지나 결과를 표시하는 방법에 대한 제안이 있는 섹션을 표시합니다.
Internationalization

RTL(오른쪽에서 왼쪽으로) 언어의 경우 검색 필드의 레이아웃이 미러링됩니다. (돋보기 아이콘 자체는 ​​미러링되지 않습니다.)
