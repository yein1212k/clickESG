Select Button

셀렉트는 사용자에게 여러 개의 옵션 목록을 제공하여 그 중 한 개의 값 혹은 다수의 값을 선택할 수 기능의 버튼입니다.

Anatomy

Options

Check box

사용자에게 여러 개의 옵션 목록을 제공하여 그 중
여러개의 값을 선택할 수 있도록 하는 경우에 사용합니다.

Checkbox group

기본적으로 체크박스 그룹은 수직입니다.

체크박스 그룹은 중형, 대형의 두 가지 크기로 제공됩니다.

Radio button

사용자가 여러 개의 옵션 중 한 개의 값 만 선택할 수 있도록 하는 경우에 사용합니다.

옵션 목록에서 새로운 옵션을 선택했을 때, 기존에 선택되었던 옵션은 자동으로 선택이 해제되어야 합니다.

Radio button layout

라디오 그룹에는 항상 레이블이 있어야 합니다.

기본적으로 라디오 그룹은 수평이지만 공간이 협소한 경우 수직으로 정렬할 수 있습니다.

Toggle button

토글은 두 가지 상태를 전환하는 요소로,
일반적으로 On/Off 상태에 사용됩니다.

Toggle button layout

레이블 높이에 맞춰 중앙에 정렬합니다.

Usage

필수 사항
  • 라디오 버튼은 필수로 한 가지는 선택되어 있어야 합니다. 필수 선택을 해야 한다면 레이블에 필수 표시를 붙여 줄 수 있습니다.
  • 체크박스 그룹은 상황에 따라 문구로 선택 사항 또는 필수 사항으로 표시할 수 있습니다.
Usage guidelines

  • 라디오 버튼 세트는 상호 배타적인 옵션 목록에서 단일 옵션만 선택하는 데
    사용해야 합니다.

  • 체크박스와 라디오 버튼은 호환되지 않습니다.

  • 라벨이 가로 공간에 비해 너무 길면 다른 줄로 줄바꿈됩니다.

  • 라디오 그룹에는 항상 옵션 목록이 무엇을 나타내는지 명확하게 설명하는
    레이블이 있어야 합니다.

Internationalization

RTL(오른쪽에서 왼쪽으로) 언어의 경우 라디오 그룹(및 해당 구성 요소)의 레이아웃이 미러링됩니다.

라디오 버튼과 아이콘은 텍스트의 오른쪽에 배치되고 텍스트는 오른쪽에
정렬됩니다.

라디오 버튼 왼쪽에 위치한 기존 레이아웃을 변경 없이 문구만 RTL을
사용하면 안됩니다.
