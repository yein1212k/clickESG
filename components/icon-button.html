Icon Button

아이콘 형태 버튼으로 공간이 충분하지 않거나 사용자가 이해하기에 충분히 직관적인 아이콘이라면
레이블 없이 아이콘만 단독으로 사용할 수 있습니다.

Anatomy

Options

GNB

해상도 공통으로 사용되는 GNB 아이콘 버튼 입니다. 다크모드와 라이드모드를 고려해 블랙과 화이트
두 가지 버전으로 사용됩니다.

Close

버튼 클릭 시 팝업 혹은 현재 페이지가 닫힙니다.

Contextual help

경험에 대한 상위 수준의 주제를 설명하고
사용자를 더 많은 정보로 안내할 수 있습니다.

Mobile tapbar

APP전용 하단 탭바에서만 사용되는 아이콘 버튼입니다.
버튼 클릭시 각 페이지로 이동합니다.

Sticky menu

본문 PC에서만 사용되는 스티키 메뉴 아이콘 버튼 입니다.
선택시 컬러 변환은 저장 버튼에만 적용되며, 뱃지는 댓글 버튼에만 적용됩니다.

Drop down arrow

드롭다운 UI에 쓰이는 아이콘으로 레이블과 함께
사용됩니다.

Player

영상 재생화면에 사용되는 아이콘 버튼입니다.

Badge

도트 배지는 두 가지 크기가 제공됩니다.
제공된 크기 중 하나를 사용하는 것이 좋습니다.

숫자 배지에는 1부터 999까지의 숫자가 표시됩니다.
업데이트 횟수가 999보다 큰 경우 '999+'가 표시됩니다.

Usage

Layout and Density

유사한 구성 요소 그룹을 구성 요소 내부에 함께 중첩하거나 단독으로 배치할 수 있습니다.
배치할 때에는 버튼 사이에는 동일한 간격이 있어야 합니다.

Usage guidelines

  • 레이블은 누구나 이해할 수 있을 만큼 간단해야 합니다.

  • 아이콘 전용 버튼과 섞어 사용하지 마세요.

  • 레이블은 명확하지 않을 수 있으므로 말줄임 사용하지 않습니다.

  • 레이블 없을 때는 각 아이콘은 의미를 명확하게 나타내야합니다.
