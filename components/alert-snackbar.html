Alert&Snackbar

사용자에게 중요한 정보 또는 페이지 전환 없이 정보를 보여줄 때 사용합니다.
PC에서는 딤드 없이 알럿 영역에만 그림자 적용하고 모바일에서는 그림자 없이 전체 화면에 딤드 적용합니다.

Anatomy

Border width
Value Usage
Alert 알림 대화 상자는 사용자가 확인해야 하는 중요한 정보를 표시합니다.
인터페이스 위에 나타나며 작업을 선택할 때까지 추가 상호 작용을 차단합니다.
Snackbar 스낵바는 일반적으로 화면 하단에 배치되어 사용자 동작에 대한 일시적인 피드백을 제공합니다.
링크 역할을 하거나 동작도 제공할 수 있습니다. 스낵바는 팝업에 비해 주의가 덜 필요한 피드백으로 사용됩니다.

Options

Title

제목은 대화 상자 상단에 굵은 글씨로 표시되며, 사용자가 작업을 계속할 경우 어떤 일이 일어날지 결과를 전달하기 위해 몇 단어를 사용합니다.

제목을 포함할지 여부를 선택할 수 있습니다.

Description

설명은 사용자가 버튼에서 제공하는 결정을 내리기 위해
알아야 하는 추가 정보나 맥락을 간략하게 전달합니다.

Confirmation variant

사용자에게 선택 사항을 확인하도록 요청하는
경우 사용합니다. 이 알림 버튼에는 어떤 조치를 취할
것인지에 대한 명확한 버튼명이 필요합니다.

Warning

경고 알림 대화 상자는 사용자에게 인식해야 할
문제와 관련하여 중요한 정보를 전달하지만,
사용자가 계속 진행하는 것을 막지는 않습니다.

Input

팝업 내에 정보 입력을 해야 하는 경우에 사용합니다.

Usage

Text overflow

제목과 설명 텍스트가 사용 가능한 수평 공간에 비해 너무 길면 다른 줄로 줄바꿈됩니다.

Overlapping buttons

세 개의 버튼을 사용하는 것은 권장되지 않습니다. 다만 필요한 경우에 따라 레벨이 다른 버튼을 적용 할 수 있습니다.

Usage guidelines

알림 대화 상자는 방해가 되므로 사용자가 확인해야 하는
중요한 정보를 표시해야 할 때 가장 좋습니다.

알림 대화 상자는 해결책에 도달할 때까지 경험을 방해하기 위한 것이므로
한 번에 하나만 표시되어야 합니다.
