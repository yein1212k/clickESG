Button Navigation

사용자가 빠르고 쉽게 접근할 수 있도록 앱의 주요 기능으로 구성되어 있습니다.
각 목적지는 아이콘과 선택적 텍스트 레이블로 표시됩니다.

Main bottom navigation

APP전용으로 디바이스 하단에 노출되는 메뉴입니다.
홈, 섹션 홈/리스트(날씨, 북한포털, 재난포털, 보도자료) 등 뉴스와 관련된 모든 리스트 영역에 노출됩니다.
로그인, 검색, 설정, 전체 메뉴, 고객센터, 로그인, 마이뉴스 등 뉴스와 관련된 영역을 제외한 일반 메뉴 화면에서는 노출되지 않습니다.

News bottom navigation

뉴스 본문에서 사용하는 네비게이션입니다.

Options

Update badge

하단 탐색 메뉴 아이콘의 배지를 사용하여 업데이트 상태를 표시할 수 있습니다.

Case

본문 전용 하단 탭바는 적용되는 브라우저 환경에 따라 제공됩니다.

Usage

Placement
  • 앱 화면의 맨 아래 부분에 배치합니다.
  • 화면에 고정되어 있으며 스크롤해도 사라지지 않습니다.
Item distribution

장치가 가로 모드일 때 하단 탐색 항목을 가로로 분산하여 화면 너비를 채울 수 있습니다.

Usage guidelines

텍스트 레이블은 줄임표나 두 줄로 표시되지 않도록
간단하게 작성해야 합니다.

하단 탐색에서 접근 가능한 메뉴는 Y뷰어를 제외하고 4개를 초과할 수
없습니다. 4개를 초과해야 하는 경우 탭을 사용합니다.
